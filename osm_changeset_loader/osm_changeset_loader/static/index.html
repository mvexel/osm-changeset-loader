Instructions

  • Go to OSM and move the map so it is centered on the area you are interested in
  • Copy the URL (https://www.openstreetmap.org/#map=...)
  • Paste the URL in the form on this page
  • Select the scale you are interested in: Neighborhood (approximately 5 km2), City (25km2) or State / Province (250 km2)
